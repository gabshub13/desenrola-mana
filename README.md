## Rodar

```bash
npm install
npm run start
```

## Teste beta local

- Expo/Metro: http://localhost:8081
- Web build local: http://localhost:4173

Para Expo Go, abra o app Expo no celular e conecte ao servidor local exibido pelo Metro.

Fluxo implementado: boas-vindas, home, busca/filtro, perfil, agendamento, pagamento simulado, chat liberado por pagamento, token de segurança, validação do token e avaliação.
